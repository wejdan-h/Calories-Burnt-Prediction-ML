# Calories Burnt Prediction Using Machine Learning

A machine learning project that predicts the number of calories burned during exercise using biological and exercise-related features.

## Dataset

- **Source:** https://www.kaggle.com/datasets/ruchikakumbhar/calories-burnt-prediction
- **Records:** 15,000
- **Target:** Calories

## Project Workflow

- Data Exploration (EDA)
- Data Preprocessing
- Data Visualization
- Feature Scaling
- Model Training
- Model Evaluation

## Models

- Linear Regression
- Decision Tree Regressor

## Results

- **Linear Regression**
  - MAE: **8.20**
  - MSE: **122.97**
  - R² Score: **0.9677**

- **Decision Tree**
  - MAE: 8.77
  - MSE: 154.27
  - R² Score: 0.9595

**Best Model:** Linear Regression
